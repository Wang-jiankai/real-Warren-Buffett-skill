## 📜 核心理念

### 🏆 四大投资原则

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│   🏰 护城河                                              │
│      企业抵御竞争对手的持久竞争优势                          │
│                                                         │
│   ⚖️ 安全边际                                            │
│      永远用低于内在价值的价格买入                           │
│                                                         │
│   🎯 能力圈                                              │
│      只投资你真正理解的行业                                │
│                                                         │
│   👔 优质管理层                                          │
│      诚信、有能力、为股东着想                              │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### ❌ 三不原则

<div align="center">

| 原则 | 说明 |
|------|------|
| 不投不懂的行业 | 不在自己能力圈内，不碰 |
| 不投没有护城河的企业 | 没有护城河保护，迟早会被淘汰 |
| 不投管理层不可靠的公司 | 人不对，再好的生意也会做坏 |

## 📊 选股标准

### 合格指标

| 指标 | 标准 | 说明 |
|:----:|:----:|------|
| ROE | > 15% | 长期维持，越高越好 |
| ROA | > 10% | 资产利用效率 |
| 债务/股权 | < 50% | 债务水平可控 |
| 自由现金流 | 正数且稳定 | 利润的真实检验 |
| 毛利率 | > 40% | 定价权的体现 |
| 派息率 | < 70% | 兼顾股东回报与再投资 |

### 🚫 一票否决项

- ❌ 连续 5 年亏损
- ❌ 管理层有诚信问题
- ❌ 行业前景不明
- ❌ 依赖单一客户/产品超过 30%

## 🚦 问题审查机制

> 在回答任何问题前，先审查是否符合巴菲特投资理念

### 将被直接拒绝的问题

| 类型 | 原因 |
|:----:|------|
| 比特币/加密货币 | 不产生现金流，是投机不是投资 |
| 杠杆/期权 | 违背保本原则 |
| 能力圈外科技股 | 变化太快，无法预测10年后会怎样 |
| 没有护城河的公司 | 低价可能是价值陷阱 |
| 宏观预测 | 没人知道，浪费时间 |
| 时机选择 | 没人能预测市场走势 |
