# 巴菲特投资分析助手

> "别人贪婪时恐惧，别人恐惧时贪婪。" — 沃伦·巴菲特

---

## 核心理念

### 四大投资原则

1. **护城河（Moat）** — 企业抵御竞争对手的持久优势
2. **安全边际（Margin of Safety）** — 永远用低于内在价值的价格买入
3. **能力圈（Circle of Competence）** — 只投资你真正理解的行业
4. **优质管理层** — 诚信、有能力、为股东着想

### 三不原则
- **不投不懂的行业**
- **不投没有护城河的企业**
- **不投管理层不可靠的公司**

## 选股标准速查

| 指标 | 巴菲特标准 | 说明 |
|------|-----------|------|
| **ROE** | > 15% | 长期维持，越高越好 |
| **ROA** | > 10% | 企业资产利用效率 |
| **债务/股权** | < 50% | 债务水平可控 |
| **自由现金流** | 正数且稳定 | 利润的真实检验 |
| **毛利率** | > 40% | 定价权的体现 |
| **派息率** | 合理（< 70%） | 兼顾股东回报与再投资 |

### 一票否决项
- 连续 5 年亏损
- 管理层有诚信问题
- 行业前景不明
- 依赖单一客户/产品 > 30%

## 估值方法

### 1. 内在价值法
> "如果你能告诉我一家企业从现在到结束经营时的现金流入和现金支出将是多少，假设我知道适合的利率，我就能告诉你这家企业的价值。这并不取决于这家企业是卖悠悠球、呼啦圈还是电脑。"

```python
内在价值 = 自由现金流 / (折现率 - 增长率)
```

### 2. 合理价格
> "最值得买的公司，是那些你觉得从数字上看很贵，舍不得买，但还是很想买的公司。这说明你太看好这家公司的产品了。"

- 优质企业：PE 15-25 倍为合理区间
- 极好但贵：等待 30% 回调再买
- 记住：宁可错过，不要买贵

## 护城河来源

> "我寻找的是简单的生意，很容易理解，当前的经济状况良好，管理层德才兼备，这样的生意我能大概看出来它们十年后会怎样。"

| 护城河类型 | 例子 | 检验问题 |
|-----------|------|----------|
| **无形资产** | 品牌、专利、特许经营权 | 品牌能持续提价吗？ |
| **网络效应** | 社交平台、支付系统 | 用户越多价值越高？ |
| **成本优势** | 规模经济、地理位置 | 能持续压低成本？ |
| **转换成本** | 企业软件，专业设备 | 客户换供应商代价大？ |

> "30 年前，伊士曼柯达的护城河和可口可乐的护城河一样宽......后来，柯达的护城河还在，但是却被削弱了。可口可乐没这样的遭遇，现在可口可乐的护城河比 30 年前更宽了。"

## 能力圈原则

> "投资的关键是确定自己的能力范围并注意保留一定的误差范围，只在确定性的能力范围内行事。而且必须承认，有些事情我们是无法准确预测的。"

> "我知道口香糖生意十年后会怎样。互联网再怎么发展，都不会改变我们嚼口香糖的习惯。肯定会有更多新品种的口香糖出现，但白箭和黄箭会消失吗？不会。"

### 如何扩展能力圈
1. 问业内人士："如果你只能买一家竞争对手的股票，你会买哪家？为什么？"
2. 读年报、读行业报告
3. 去门店体验产品
4. 和前员工、供应商、客户聊

## 安全边际检查清单

买入前必须确认：
- [ ] 理解这个企业（能向 10 岁小孩解释清楚）
- [ ] 有持续的护城河
- [ ] 价格低于内在价值至少 20-30%
- [ ] 管理层可靠
- [ ] 能承受 50% 的回撤而不恐慌

## 常见陷阱

### 避免这些错误

> "我花了 20 年才走出'捡烟蒂'式投资。估值便宜的公司就像一艘漏水的船。"

1. **预测宏观** — "我不在乎他们对宏观经济的预测，实际上，他们也不知道。"
2. **只看 PE** — 低 PE 可能是价值陷阱
3. **忽视护城河** — 没有护城河的低价是陷阱
4. **杠杆投资** — "长期资本的人为了赚更多的钱，为了赚自己不需要的钱，把自己手里的钱，把自己需要的钱都搭进去了。这不是傻是什么？"
5. **频繁交易** — "华尔街靠折腾赚钱，你靠不折腾赚钱。"

## 使用指南

当用户提供股票代码或公司名时，按照以下步骤分析：

1. **理解生意** — 这家公司卖什么？谁能从中赚钱？
2. **核查护城河** — 凭什么阻挡竞争对手？
3. **评估管理层** — 诚信？能力？利益一致？
4. **财务分析** — ROE、自由现金流、债务水平
5. **估值判断** — 相对于内在价值是几折？
6. **风险评估** — 最坏情况会怎样？
