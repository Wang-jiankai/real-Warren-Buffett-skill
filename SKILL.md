# 巴菲特投资分析助手

> "别人贪婪时恐惧，别人恐惧时贪婪。" — 沃伦·巴菲特

## 核心理念

### 四大投资原则
1. **护城河（Moat）** — 企业抵御竞争对手的持久优势
2. **安全边际（Margin of Safety）** — 永远用低于内在价值的价格买入
3. **能力圈（Circle of Competence）** — 只投资你真正理解的行业
4. **优质管理层** — 诚信、有能力、为股东着想

### 三不原则
- **不投不懂的行业**
- **不投没有护城河的企业**
- **不投管理层不可靠的公司**

## 选股标准速查

| 指标 | 巴菲特标准 | 说明 |
|------|-----------|------|
| **ROE** | > 15% | 长期维持，越高越好 |
| **ROA** | > 10% | 企业资产利用效率 |
| **债务/股权** | < 50% | 债务水平可控 |
| **自由现金流** | 正数且稳定 | 利润的真实检验 |
| **毛利率** | > 40% | 定价权的体现 |
| **派息率** | 合理（< 70%） | 兼顾股东回报与再投资 |

### 一票否决项
- 连续 5 年亏损
- 管理层有诚信问题
- 行业前景不明
- 依赖单一客户/产品 > 30%

## 估值方法

### 1. 内在价值法
```
内在价值 = 自由现金流 / (折现率 - 增长率)
```

**简化版：** 用 10 年自由现金流折现估算

### 2. PEG 估值
```
PEG = 市盈率 / 预期增长率
PEG < 1 = 可能被低估
```

### 3. 合理价格
- 优质企业：PE 15-25 倍为合理区间
- 极好但贵：等待 30% 回调再买
- 记住：宁可错过，不要买贵

## 企业分析框架（15问）

### 行业与竞争（1-5）
1. **护城河来源是什么？** 品牌、专利、网络效应、成本优势、转换成本？
2. **行业竞争格局如何？** 是寡头、分散还是一家独大？
3. **潜在威胁？** 新进入者、替代品、技术颠覆？
4. **行业增长空间？** 天花板在哪？
5. **行业周期性？** 强周期还是弱周期？

### 财务健康（6-10）
6. **ROE 趋势？** 稳定 > 15% 还是波动大？
7. **债务水平？** 能否承受 3 年低营收？
8. **自由现金流？** 是否匹配净利润？
9. **盈利质量？** 靠主业还是一次性收益？
10. **历史毛利率？** 稳定还是持续下降？

### 管理层（11-13）
11. **管理层诚信？** 过往是否有失信记录？
12. **资本配置能力？** 回购、分红、投资决策是否合理？
13. **激励机制？** 是否与股东利益一致？

### 估值（14-15）
14. **当前价格？** 相对内在价值是几折？
15. **机会成本？** 这笔钱投别的是否更好？

## 安全边际检查清单

买入前必须确认：
- [ ] 理解这个企业（能向 10 岁小孩解释清楚）
- [ ] 有持续的护城河
- [ ] 价格低于内在价值至少 20-30%
- [ ] 管理层可靠
- [ ] 能承受 50% 的回撤而不恐慌

---

## 常见陷阱

### 避免这些错误
1. **预测未来增长** — 增长最难预测
2. **只看 PE** — 低 PE 可能是价值陷阱
3. **忽视债务** — 债务可以在危机时杀死企业
4. **过度分散** — 10 个平庸不如 3 个优秀
5. **频繁交易** — 手续费和税收会吃掉利润
