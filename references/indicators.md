# 关键财务指标解读

## 一、盈利能力指标

### ROE（净资产收益率）
```python
ROE = 净利润 / 股东权益 × 100%
```

| ROE 区间 | 判断 |
|---------|------|
| > 25% | 优秀，强大的竞争优势 |
| 15-25% | 良好，有护城河 |
| 10-15% | 一般，需要分析原因 |
| < 10% | 差，可能有隐藏问题 |

**注意：** 极高 ROE（> 50%）要警惕，可能是负债过高或一次性收益

**杜邦分解：**
```
ROE = 净利润率 × 资产周转率 × 权益乘数
    = (净利/收入) × (收入/资产) × (资产/权益)
```

### ROA（资产收益率）
```python
ROA = 净利润 / 总资产 × 100%
```

- > 10% = 优质
- 5-10% = 合格
- < 5% = 差（资产效率低）

### ROIC（投入资本回报率）
```python
ROIC = 税后营业利润(EBIT) / 投入资本
投入资本 = 股东权益 + 有息负债 - 现金
```

- ROIC > WACC（加权平均资本成本）= 创造价值
- ROIC < WACC = 破坏价值

## 二、负债能力指标

### 资产负债率
```python
资产负债率 = 总负债 / 总资产 × 100%
```

| 比率 | 判断 |
|------|------|
| < 30% | 保守，财务稳健 |
| 30-50% | 可接受 |
| 50-70% | 偏高，需关注 |
| > 70% | 危险，高杠杆 |

**行业差异：**
- 金融：负债率天然高（> 90% 正常）
- 消费品：通常 < 50%
- 重资产：需结合现金流判断

### 流动比率
```python
流动比率 = 流动资产 / 流动负债
```

- > 2 = 良好
- 1.5-2 = 可接受
- < 1 = 短期偿债风险

### 利息覆盖倍数
```python
利息覆盖 = EBIT / 利息支出
```

- > 5 = 安全
- 3-5 = 勉强可以
- < 3 = 危险

## 三、现金流指标

### 自由现金流（FCF）
```python
FCF = 经营现金流 - 资本支出
```

**优质企业特征：**
- FCF 持续为正
- FCF / 净利润 > 1（利润真实）
- 资本支出稳定且合理

### 现金流肖像
| 类型 | 经营 | 投资 | 筹资 | 说明 |
|------|------|------|------|------|
| **妖精型** | + | - | + | 靠筹资生存，危险 |
| **老母鸡型** | + | - | - | 稳定收息，适合价值投资 |
| **疯牛型** | + | - | + | 高增长，靠筹资扩张 |
| **毁灭型** | - | +/- | + | 烧钱模式 |

**巴菲特偏好：老母鸡型**

## 四、估值指标

### PE（市盈率）
```python
PE = 每股市价 / 每股收益(EPS)
```

| PE 区间 | 判断 |
|---------|------|
| < 10 | 便宜（可能是陷阱） |
| 10-20 | 合理偏低 |
| 20-30 | 合理偏高 |
| > 30 | 贵（高增长预期） |

**使用方法：**
- 同行业对比（银行 PE < 科技 PE）
- 历史 PE 对比（当前 vs 过去 10 年均值）
- 预期增长率修正（PEG）

### PEG 比率
```python
PEG = PE / 年化增长率 × 100
```

| PEG | 判断 |
|-----|------|
| < 1 | 可能被低估 |
| 1-2 | 合理 |
| > 2 | 可能被高估 |

### PB（市净率）
```python
PB = 每股市价 / 每股净资产
```

- < 1 = 市价低于净资产
- 1-3 = 正常区间
- > 3 = 溢价（品牌/预期）

**适用：** 金融、能源、重资产行业
**不适用：** 轻资产、科技公司

### PS（市销率）
```python
PS = 总市值 / 营业收入
```

| PS | 判断 |
|----|------|
| < 1 | 便宜 |
| 1-3 | 合理 |
| > 10 | 贵（高增长预期） |

## 五、股息指标

### 股息率
```python
股息率 = 每股股息 / 每股股价 × 100%
```

- > 5% = 高股息
- 3-5% = 良好
- 1-3% = 一般
- < 1% = 成长型，不靠股息

### 派息率
```python
派息率 = 每股股息 / EPS × 100%
```

| 派息率 | 判断 |
|--------|------|
| < 30% | 低，保留现金再投资 |
| 30-70% | 合理平衡 |
| > 70% | 高，难以持续 |
| > 100% | 危险（吃老本） |

---

## 六、成长指标

### 营收增长率
```python
营收增长率 = (本期营收 - 上期营收) / 上期营收 × 100%
```

### 净利润增长率
- 稳定 > 15% = 优质成长
- 波动大 = 需要分析原因

### 复合增长率（CAGR）
```python
CAGR = (期末值/期初值)^(1/年数) - 1
```

## 七、综合打分卡

| 指标 | 优秀 | 合格 | 差 |
|------|------|------|-----|
| ROE | > 20% | 15-20% | < 15% |
| 资产负债率 | < 40% | 40-60% | > 60% |
| 流动比率 | > 2 | 1.5-2 | < 1.5 |
| 自由现金流 | 正且稳定 | 偶尔为负 | 长期为负 |
| PE（相对行业） | 低于均值 | 均值附近 | 高于均值 |
| 股息率 | 3-5% | 1-3% | < 1% |
