# 关键财务指标解读

## 一、盈利能力指标

### ROE（净资产收益率）
> "ROE 低于 15% 的公司不值得投资。"

```python
ROE = 净利润 / 股东权益 × 100%
```

| ROE 区间 | 判断 |
|---------|------|
| > 25% | 优秀，强大的竞争优势 |
| 15-25% | 良好，有护城河 |
| 10-15% | 一般，需要分析原因 |
| < 10% | 差，可能有隐藏问题 |

**杜邦分解：**
```
ROE = 净利润率 × 资产周转率 × 权益乘数
    = (净利/收入) × (收入/资产) × (资产/权益)
```

### ROA（资产收益率）
```python
ROA = 净利润 / 总资产 × 100%
```

- > 10% = 优质
- 5-10% = 合格
- < 5% = 差（资产效率低）

### ROIC（投入资本回报率）
```python
ROIC = 税后营业利润(EBIT) / 投入资本
投入资本 = 股东权益 + 有息负债 - 现金
```

- ROIC > WACC = 创造价值
- ROIC < WACC = 破坏价值

## 二、盈利质量

### 经营现金流 vs 净利润
> "我们确切知道的是，喜诗每年能卖出 1600 万磅糖果，每磅售价 1.95 美元，每磅利润 0.25 美元。"

```python
盈利质量 = 经营现金流 / 净利润
```

| 比值 | 判断 |
|------|------|
| > 1 | 优质，利润真实 |
| ≈ 1 | 正常 |
| < 1 | 需警惕，可能有水分 |

### 盈利稳定性
- 过去 5-10 年 ROE 是否稳定？
- 是否依赖一次性收益？
- 周期性强不强？

## 三、负债能力指标

### 资产负债率
> "资产负债率应该以不超过 45% 为宜。"

```python
资产负债率 = 总负债 / 总资产 × 100%
```

| 比率 | 判断 |
|------|------|
| < 40% | 保守，财务稳健 |
| 40-50% | 可接受 |
| 50-70% | 偏高，需关注 |
| > 70% | 危险，高杠杆 |

**行业差异：**
- 金融：负债率天然高（> 90% 正常）
- 消费品：通常 < 50%
- 重资产：需结合现金流判断

### 流动比率
```python
流动比率 = 流动资产 / 流动负债
```

- > 2 = 良好
- 1.5-2 = 可接受
- < 1 = 短期偿债风险

### 利息覆盖倍数
```python
利息覆盖 = EBIT / 利息支出
```

- > 5 = 安全
- 3-5 = 勉强可以
- < 3 = 危险

## 四、现金流指标

### 自由现金流（FCF）
> "自由现金流是评估公司的关键。一家公司的价值取决于其现金流折现。"

```python
FCF = 经营现金流 - 资本支出
```

**优质企业特征：**
- FCF 持续为正
- FCF / 净利润 > 1（利润真实）
- 资本支出稳定且合理

### 现金流肖像
| 类型 | 经营 | 投资 | 筹资 | 说明 |
|------|------|------|------|------|
| **妖精型** | + | - | + | 靠筹资生存，危险 |
| **老母鸡型** | + | - | - | 稳定收息，适合价值投资 |
| **疯牛型** | + | - | + | 高增长，靠筹资扩张 |
| **毁灭型** | - | +/- | + | 烧钱模式 |

**巴菲特偏好：老母鸡型**

## 五、估值指标

### PE（市盈率）
```python
PE = 每股市价 / 每股收益(EPS)
```

> "最理想的情况是，买的时候，你觉得根本不会有这一天。"

| PE 区间 | 判断 |
|---------|------|
| < 10 | 便宜（可能是陷阱） |
| 10-20 | 合理偏低 |
| 20-25 | 合理偏高 |
| > 25 | 贵（需高增长支撑） |

**注意：** PE 必须结合护城河和增长率看

### PEG 比率
```python
PEG = PE / 年化增长率 × 100
```

| PEG | 判断 |
|-----|------|
| < 1 | 可能被低估 |
| 1-2 | 合理 |
| > 2 | 可能被高估 |

### 市净率（PB）
```python
PB = 每股市价 / 每股净资产
```

适用：金融、能源、重资产行业
不适用：轻资产、科技公司

## 六、股息指标

### 股息率
```python
股息率 = 每股股息 / 每股股价 × 100%
```

| 股息率 | 判断 |
|--------|------|
| > 5% | 高股息（需确认可持续性） |
| 3-5% | 良好 |
| 1-3% | 一般 |
| < 1% | 成长型，不靠股息 |

### 派息率
```python
派息率 = 每股股息 / EPS × 100%
```

| 派息率 | 判断 |
|--------|------|
| < 30% | 低，保留现金再投资 |
| 30-70% | 合理平衡 |
| > 70% | 高，难以持续 |
| > 100% | 危险（吃老本） |

---

## 七、护城河指标

### 毛利率
> "我们选择那些不需要持续资本投入却能产生自由现金流的生意。"

```python
毛利率 = (收入 - 成本) / 收入 × 100%
```

| 毛利率 | 判断 |
|--------|------|
| > 60% | 极强，有定价权 |
| 40-60% | 良好，有护城河 |
| 20-40% | 一般，竞争激烈 |
| < 20% | 差，可能没有护城河 |

### 净利率趋势
- 稳定/上升 = 护城河在加强
- 下降 = 护城河在削弱

### 资产周转率
```python
资产周转率 = 收入 / 总资产
```

- 轻资产公司：周转率通常较低，但利润率高
- 重资产公司：需要高周转率才能有好 ROE

## 八、综合打分卡

| 指标 | 优秀 | 合格 | 差 |
|------|------|------|-----|
| ROE | > 20% | 15-20% | < 15% |
| 资产负债率 | < 40% | 40-60% | > 60% |
| 流动比率 | > 2 | 1.5-2 | < 1.5 |
| 毛利率 | > 40% | 20-40% | < 20% |
| 自由现金流 | 正且稳定 | 偶尔为负 | 长期为负 |
| 净利率趋势 | 上升 | 稳定 | 下降 |

## 九、估值步骤

### 第一步：估算内在价值
```python
内在价值 = 未来10年自由现金流之和
```

### 第二步：确定安全边际
> "安全边际是你购买时获得的折扣。"

| 折扣 | 建议 |
|------|------|
| < 20% | 偏贵 |
| 20-40% | 合理，有安全边际 |
| > 40% | 深度价值，可能有陷阱 |

### 第三步：机会成本比较
- vs 国债收益率
- vs 指数基金历史回报
- vs 其他确定性投资

## 十、利率与估值

> "利率之于资产的估值，好比引力之于质量。"

### 利率对估值的影响
- 低利率 = 资产估值可以更高
- 高利率 = 资产估值必须下降
- 利率上升 = 股票估值下压

### 当前市场判断
- 30年期国债收益率接近历史低位
- 股票相对于债券仍然便宜（如果利率维持低位）
- **关键问题**：未来利率走向？

---

## 十一、回购与分红

### 回购的意义
> "回购就是把收益返还给股东的方式，和分红本质上是一样的。"

### 判断标准
- 回购价格 < 内在价值 = 创造股东价值
- 回购价格 > 内在价值 = 损害股东价值
- 管理层回购决策是否明智？

### 分红政策
- 稳定或增长的分红 = 好信号
- 过度分红影响再投资 = 需要分析
- **政府不应规定企业分红政策**
