# 企业分析提问清单

## 一、行业与竞争格局

### 1. 护城河来源
企业靠什么阻挡竞争对手？

| 类型 | 例子 | 检验问题 |
|------|------|----------|
| **无形资产** | 品牌、专利、特许经营权 | 品牌能持续提价吗？专利到期风险？ |
| **网络效应** | 社交平台、支付系统 | 用户越多价值越高？ |
| **成本优势** | 规模经济、地理位置 | 能持续压低成本？ |
| **转换成本** | 企业软件、专业设备 | 客户换供应商代价大？ |

### 2. 竞争格局分析
```
问题：这个行业是赢者通吃还是百花齐放？
- 行业集中度（CR3/CR5）
- 主要竞争对手是谁
- 差异化程度如何
```

### 3. 五大威胁
1. **新进入者** — 壁垒够高吗？
2. **替代品** — 有更好的解决方案吗？
3. **供应商议价** — 依赖单一供应商？
4. **客户议价** — 客户能轻易压价？
5. **行业内卷** — 价格战风险？

### 4. 行业生命周期
- 导入期：高增长、高风险、烧钱阶段
- 成长期：快速增长、竞争加剧
- 成熟期：稳定增长、格局稳定
- 衰退期：负增长、收缩

### 5. 周期性判断
```
强周期：钢铁、煤炭、航空、房地产
弱周期：食品、医药、消费必需品
```

## 三、财务分析

### 9. 盈利能力
```python
# 核心指标
ROE = 净利润 / 股东权益
ROA = 净利润 / 总资产
ROIC = 税后营业利润 / 投入资本

# 盈利质量
经营现金流 / 净利润 > 1 = 利润真实
```

### 10. 债务健康
```python
# 短期偿债
流动比率 = 流动资产 / 流动负债 > 1.5
速动比率 = (流动资产-存货) / 流动负债 > 1

# 长期债务
资产负债率 = 总负债 / 总资产 < 50%
利息覆盖 = EBIT / 利息支出 > 3
```

### 11. 现金流
```
自由现金流 = 经营现金流 - 资本支出

检验标准：
- FCF > 净利润（优质）
- FCF ≈ 净利润（正常）
- FCF < 净利润（需警惕）
```

## 四、管理层评估

### 12. 诚信记录
- 财务报表是否真实（无会计操纵）
- 过往承诺是否兑现
- 并购历史（是否乱收购）

### 13. 资本配置
| 行为 | 判断 |
|------|------|
| 回购股票 | PE 低 + 现金多时为佳 |
| 现金分红 | 找不到好投资时合理 |
| 收购扩张 | 警惕高溢价并购 |
| 保留现金 | 等待机会 |

### 14. 激励相容
- 管理层薪酬与股东利益是否一致
- 是否有股权激励（稀释效应）
